# Design System: Laca Self-Ordering KDS

## 1. Visual Theme & Atmosphere
Retro festival counter meets clean utility. The interface should feel like a small printed menu card from an old neighborhood stall: warm paper, dark ink, restrained accent color, and sturdy components that are easy to tap under pressure. The aesthetic is minimal, not nostalgic clutter.

## 2. Color Palette & Roles
- **Roasted Ink (#251914):** Main text, primary buttons, strong borders.
- **Rice Paper (#fff8e8):** Main surfaces and page background.
- **Aged Card (#fffdf5):** Food cards, admin panels, KDS order cards.
- **Burnt Mandarin (#d7652c):** Primary accent, active highlights, add buttons.
- **Olive Paid (#496f38):** Paid/completed positive states.
- **Ledger Red (#a83c2d):** Cancel/destructive states.
- **Faded Sepia (#8a725d):** Secondary copy and helper text.
- **Rule Line (#d9c7aa):** Subtle dividers, form strokes, card borders.

## 3. Typography Rules
Use Quicksand across customer and admin interfaces so the plugin feels like one product. Body text sits around 15-16px for readability on iPad/mobile admin use. Headings are compact and warm, usually 500-600 weight, not heavy poster type. Labels may use small uppercase with light letter spacing, but avoid overusing bold text.

## 4. Component Stylings
- **Buttons:** Pill-shaped, high contrast, minimal gradients. Primary actions use Roasted Ink with Rice Paper text; secondary actions use paper background with ink borders.
- **Cards/Containers:** Warm paper surfaces, 1px sepia borders, soft diffused shadows. Corners are rounded but not bubbly.
- **Inputs/Forms:** Calm ivory background, visible 1px borders, focus states in Burnt Mandarin.
- **Status Badges:** Small rounded labels with low-saturation fills and clear text contrast.

## 5. Layout Principles
Prioritize speed and clarity. Use generous spacing, strong section separation, and compact grids. On mobile, keep cart and checkout actions thumb-friendly. Avoid visual noise; every decorative mark should support the printed-menu feeling.
